Introduction to HTML

HTML(Hypertext Markup Language) is used to create document on the World Wide Web. It is simply a collection a certain key words called "Tags" that are helpful in writing the document to be displayed using a browser on Internet. HTML was depeloped by Tim Berners-Lee in 1992.
